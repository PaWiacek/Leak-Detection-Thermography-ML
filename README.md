# Analysis of Gas Leak Detection Using Thermography and Deep Learning
Information encoded in the filename
FLIRXXXX_<background>_<surface>_<leak>_<height>_<phi>_<theta>.png


Field description:

FLIRXXXX – recording number

labo_BG / screen_BG – background type

matte / reflex – test tube surface type

leak / noleak – leak presence

Z1 / Z2 / Z3 – tripod height

phi0 / phi-30 / phi30 – sample orientation (degrees)

theta-10 / theta-5 / theta0 / theta5 / theta10 – camera angle (degrees)

##############################################################

Example
FLIR0199_labo_BG_matte_leak_Z3_phi30_theta0.png


Meaning:

recording number: 0199

background: laboratory

surface: matte

leak: present

tripod height: Z3

sample orientation: 30 degree

camera angle: 0 degree
