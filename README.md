# Analysis of Gas Leak Detection Using Thermography and Deep Learning

## Overview
This project focuses on gas safety classification using thermal camera-derived data and convolutional neural networks CNN. The goal was to distinguish between leak and no-leak classes based on experimental measurements, as well as to assess the influence of reflective tube surface and thermal noise in the backround.
